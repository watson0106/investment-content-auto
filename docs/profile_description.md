# noteプロフィール文 改善ドラフト

note管理画面 → 設定 → プロフィール の「自己紹介」欄に貼り付け。

---

## 案A（実績訴求型・推奨）

兼業投資家（投資歴○年）／個別株でインデックスをアウトパフォームしたい人のためのnoteを書いています。

毎朝7時、その日に動きそうな注目銘柄を「**買い水準・損切り・利確目安まで具体的な数字付き**」で配信。週1で**実トレードの全記録**（先週の損益・勝率・反省点）も公開しています。

「読んで終わり」を脱したい人向け。

▶ メンバーシップ¥980/月・初月無料で全記事読み放題

---

## 案B（共感訴求型）

ニュースを読んでも「で、私はどう動けばいいのか」が分からない。  
そんなモヤモヤを抱える兼業投資家のためにnoteを書いています。

毎朝、その日のニュースを「私ならどう動くか」まで翻訳して配信。  
個別株でインデックスをアウトパフォームすることが目標です。

毎週月曜には**実トレード結果を全公開**しています（勝ちも負けも隠さず）。

▶ メンバーシップ¥980/月・初月無料

---

## 案C（短く・直球）

毎朝7時、その日の注目銘柄2本を「買値・損切り・利確」まで具体的な数字付きで配信。  
週1で実トレード全公開。  
個別株でインデックスを超えたい人へ。  
メンバーシップ¥980/月・初月無料。

---

## 推奨は案A

理由:
- 「○年」と書くだけで信頼感が一段階上がる（具体的な数字を入れる）
- 「数字付き」「全公開」というキーワードで他のnote投資家と差別化
- 最後に明確なCTA（メンバーシップ価格と無料期間）

採用する案を決めて、note管理画面に貼り付けてください。

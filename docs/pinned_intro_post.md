## 本文

### なぜこの note を読んでほしいのか

はじめまして。投資 note は世の中に山ほどありますが、その中で私のnoteを読む価値が**何か明確にあるべきだ**と思うので、最初にそれを書きます。

世の中の投資 note の9割は次のどれかです:

- 「○○が上がる！」と煽って、有料記事を買うと「リスクに注意」で終わる
- 過去のチャートを後付けで解説する「答え合わせ系」
- 暗号資産・FX のスキャルピングのみで個別株は触らない
- 経済評論家風で、自分のポジションは絶対に出さない

私はこれらが全部嫌いだから、**真逆をやっています**。

---

### 私のスタンス

1. **数字でしか語らない**
   「高め」「下落」「大きく」とは書かない。「+3.5%」「-1.7%」「PF1.85」と書きます。

2. **自分のポジションを毎週開示する**
   先週何を買って、いくらで決済して、いくら勝った/負けたか。隠しません。

3. **負けトレードもそのまま出す**
   勝率57%ということは43%は負けます。負けの理由も同じ深さで書きます。

4. **「絶対儲かる」とは書かない**
   過去2年OOS検証で勝率57%・PF1.85のエッジを確認していますが、未来は分かりません。それでも「いつ・いくらで・なぜ」を毎日数字で出すことに意味があると思っています。
